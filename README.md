# Kifusuru
Kifusuru merupakan aplikasi berbasis web yang dirancang untuk membantu pengguna mengevaluasi kesiapan keuangan dalam menghadapi situasi darurat. Melalui analisis ketahanan finansial, simulasi risiko, dan rekomendasi strategi perlindungan, Kifusuru membantu pengguna membangun jaring pengaman keuangan yang lebih kuat.
